# VoltaWebPage MVP

## Frontend

```bash
cd frontend
npm install
npm run dev
```

## Backend

```bash
cd backend
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload
```

Frontend: `http://127.0.0.1:3000/tools/low-high-cut`

Backend: `http://127.0.0.1:8000`
