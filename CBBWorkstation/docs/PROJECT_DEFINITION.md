# Project Definition

## What C.B.B Workstation Is

C.B.B Workstation is an interactive accompaniment workstation for beginner musicians. Users enter chord progressions, choose a backing style, switch arranger-style song sections, and practice along with generated backing grooves.

The product goal is to make accompaniment practice feel structured and musical without exposing users to the complexity of a full DAW.

## Why It Exists

Beginner musicians often struggle with three things:

- turning chord ideas into something playable
- practicing timing and section changes with musical context
- exploring accompaniment without deep production knowledge

C.B.B Workstation addresses those problems by presenting a guided workstation-style interface with a small set of meaningful controls instead of a large production environment.

## Product Position

- Beginner-friendly accompaniment workstation
- Practice-focused desktop tool
- Not a DAW replacement
- Not dependent on any proprietary Yamaha format or workflow

## MVP Intent

The MVP proves that the core interaction model is understandable:

1. type chords
2. choose a style
3. switch sections like an arranger
4. shape groove feel
5. start and stop playback

If users understand and respond well to this interaction model, playback and musical generation can be expanded in later milestones.
